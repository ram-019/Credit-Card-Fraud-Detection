2. Feature Selection¶

The phase that will be performed in this report is Features Selection.

It is the process in which attributes are selected that can predict a variable more accurately, thus eliminating the irrelevant ones that diminish the value of the accuracy and the quality of the model itself.

The first step that will be performed is the Correlation Matrix between the continuous variables, so without the target variable (Class).
